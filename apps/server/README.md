# server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.2. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Development

Remember to run the following command to update the OpenAPI spec for the frontend 
to use the newest API interface.

```zsh
bun run openapi
```
